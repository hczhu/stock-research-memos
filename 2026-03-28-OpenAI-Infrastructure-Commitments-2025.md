# OpenAI's $1.15T Infrastructure Commitments

OpenAI spent 2025 breaking up with Microsoft. The exclusivity ended in January [1]. What followed was a lot of contracts.

$1.15 trillion (!) in total commitments across seven vendors:
- Broadcom: $350B - custom AI chips co-designed with OpenAI [2]
- Oracle: $300B - five years, starting 2027, part of Stargate [3]
- Microsoft Azure: $250B - still the biggest single relationship [4]
- Nvidia: $100B - GPUs across 10+ GW of data centers [5]
- AMD: $90B - Instinct GPUs, 6 GW [6]
- AWS: $38B - seven years, Nvidia clusters [7]
- CoreWeave: $22.4B - neocloud, GPU-as-a-service [8]

I had to double-check the Broadcom number. $350B for custom chips from a single vendor. But most of this is forward-looking. Oracle doesn't start until 2027. Broadcom and AMD deploy H2 2026. Today, OpenAI still runs mostly on Azure and CoreWeave.
