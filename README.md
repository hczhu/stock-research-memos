# stock-research-memos

Personal investment research memos. Raw notes, data extractions, and analysis on individual companies, sectors, and macro themes — primarily AI infrastructure, semiconductors, SaaS, and China tech.

## Structure

```
memos/YYYY-MM-DD-topic.md
```

All memos live flat in `memos/`. Files are dated at creation and tagged with `[[WikiLink]]` syntax compatible with Logseq and Obsidian.

## Format

Each memo opens with a tags line and a thesis or source attribution:

```markdown
- tags:: [[NVIDIA]], [[HBM]], [[semiconductor]]

- ## Title
	- **Source**: SemiAnalysis, May 2026
	- **Thesis**: one-line investment takeaway
```

Memos typically contain some combination of: structured data tables extracted from screenshots or reports, quantitative data points with explicit source attribution, investment implications, and open questions. Some memos include an `## X Post` section with a ready-to-publish write-up.
