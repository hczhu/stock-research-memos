- ## Users and Engagement
	- **Four-dimensional growth strategy**: all ages, international, Roblox everywhere, vibrant economy

	- International growth:
		- Japan DAUs **+45% YoY**
		- India DAUs **+59% YoY**
		- Growth supported by AI-powered language translation and safety/civility foundation
	- **Roblox Cloud**: 140,000+ servers across 19 cities in North America, Asia, and Europe
		- One server hosts multiple instances; each instance allows up to 200 concurrent users
		- High-speed databases, scalable object storage, and message queuing run on AWS
